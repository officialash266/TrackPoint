TRACKPOINT DATABASE LOCATION
============================

The SQLite database is stored inside this project folder:

data\trackpoint.db

Full example:
C:\Users\YourName\Desktop\TrackPoint_EXE_Project_DB\data\trackpoint.db

The SQL backup is stored here:

data\backup.sql

IMPORTANT
---------
- Do not delete the data folder when rebuilding the EXE.
- BUILD_TRACKPOINT_EXE.bat deliberately keeps the data folder.
- TrackPoint.exe is built into the project root, beside main.py.
- The source-code version and EXE version use the same database.
- Double-click OPEN_DATABASE_FOLDER.bat to open the database folder.
